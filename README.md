# kel-circle

Synchronized multi-KEL circle protocol built on
[KERI](https://weboftrust.github.io/ietf-keri/draft-ssmith-keri.html).

## Documentation

- [Home](https://paolino.github.io/kel-circle/)
- [Base Semantics](https://paolino.github.io/kel-circle/base-semantics/)
- [Event Classification](https://paolino.github.io/kel-circle/event-classification/)

## Quick start

```bash
nix develop --quiet -c just ci
```
